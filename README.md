## Local Development

**Start the server**
```bash
bundle exec jekyll serve --livereload
```

Site runs at http://localhost:4000/brewinghood/ — LiveReload refreshes the browser automatically on file saves.

**Stop the server**

`Ctrl+C` in the terminal running the server.

> Gemfile and all local dev files are gitignored and do not need to be committed.
